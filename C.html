Chat

El chat es una forma de comunicación digital en tiempo real, que permite a dos o más personas intercambiar mensajes de texto instantáneamente a través de una red, como Internet o una intranet.

Cracker

Un cracker es una persona que viola la seguridad de sistemas informáticos con el propósito de causar daño, robar información, sabotear sistemas o eludir mecanismos de protección como contraseñas, licencias de software o cifrado

Chip

Un chip, también conocido como circuito integrado (CI), es un componente electrónico diminuto fabricado en una pequeña lámina de material semiconductor (usualmente silicio), que contiene millones o incluso miles de millones de transistores. Estos transistores permiten realizar operaciones electrónicas como procesamiento, almacenamiento o control de señales.

Compilador

Un compilador es un programa informático que traduce el código fuente escrito en un lenguaje de programación de alto nivel (como C, Java o Pascal) a código máquina o lenguaje ensamblador, que puede ser entendido y ejecutado directamente por la computadora.

Comprimir

Comprimir en informática es el proceso de reducir el tamaño de un archivo o conjunto de archivos, utilizando algoritmos que eliminan redundancias o codifican la información de manera más eficiente. El objetivo principal es ahorrar espacio de almacenamiento y facilitar la transferencia de datos por redes.

Cookie

Una cookie es un pequeño archivo de texto que un sitio web almacena en el navegador del usuario cuando este lo visita. Su función principal es guardar información sobre la navegación del usuario, como preferencias, inicio de sesión, idioma, o el contenido de un carrito de compras.

CPU

La CPU es el componente principal de una computadora encargado de ejecutar instrucciones y procesar datos. Se le considera el "cerebro" del sistema, ya que realiza operaciones lógicas, aritméticas y de control según lo especificado por los programas.
